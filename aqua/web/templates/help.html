Getting Started with Aqua

Welcome to Aqua IoT Security Platform

Aqua is a comprehensive security platform designed to protect your IoT devices from various threats. This guide will help you get started with the platform and make the most of its features.

Step 1: Set up your environment

Before you begin using Aqua, ensure that you have completed the installation and configuration steps:

  • Install the required dependencies
  • Configure network settings
  • Set up user accounts and permissions
For detailed installation instructions, refer to the Installation Guide.
Step 2: Discover devices

The first task is to discover IoT devices on your network:

  1. Navigate to the Devices page
  2. Click on the Add Device button
  3. Select Auto-Discover to scan your network
  4. Alternatively, manually add devices by providing their details
Step 3: Run your first scan

Once you've added your devices, it's time to run a security scan:

  1. Go to the Dashboard page
  2. Click on the New Scan button
  3. Select the target network or specific devices
  4. Choose the scan type (Quick or Full)
  5. Click Start Scan to begin
Step 4: Review results and take action

After the scan completes, you'll be able to review the results:

  • Check the Vulnerabilities page to see identified security issues
  • Set up alerts for critical issues
  • Generate reports for your team
  • Implement recommended remediation steps

Frequently Asked Questions

To scan your network for IoT devices, navigate to the Devices page and click the Scan Network button in the top-right corner. Enter your network range (e.g., 192.168.1.0/24) and select the scan type.

The scan will begin and display all discovered devices once complete.

To generate security reports, navigate to the Reports page and click the New Report button. Select the report type, specify the parameters, and click Generate Report.

You can also schedule recurring reports by clicking the Schedule button.

To configure alert notifications, go to the Settings page and select the Notifications tab. Enable the notification methods you prefer (email, SMS, webhooks) and specify the recipients.

You can customize which types of alerts trigger notifications by adjusting the alert severity thresholds.
